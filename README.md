# Filmų katalogo sistema

## Tikslas
Sukurti C++ programą filmų duomenų valdymui.

## Funkcionalumas
- Peržiūrėti filmus
- Pridėti filmą
- Redaguoti filmą
- Ištrinti filmą
- Ieškoti pagal pavadinimą
- Rikiuoti pagal reitingą

## Duomenų failas
movies.txt – saugo filmų informaciją:
ID;Pavadinimas;Metai;Reitingas

## Paleidimas
1. Atidaryti project C++ aplinkoje
2. Kompiliuoti
3. Paleisti programą

## Papildomos funkcijos
- Paieška pagal pavadinimą
- Rikiavimas pagal reitingą
